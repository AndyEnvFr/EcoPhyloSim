equilibriumCheck

This is the analysis of 100 simulation runs. These runs where conducted with a grid size of 50 by 50 cells. Each simulation was run over 100k generations with the same settings. The only thing different in each run was the random seed. This analysis is designed to assess the variation between simulation runs and to determine whether the model reaches equilibrium.
